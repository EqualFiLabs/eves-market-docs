## Integration notes

Call market functions through the diamond. Treat standalone contracts as explicit token, vault, lending, faucet, or router integrations.

EveUSD is 18 decimals even when the backing token is 6 decimal USDC. The current testnet EVE token is also the vote token; there is no separate EVE votes token in this deployment.

Market-side orderbook IDs are deterministic, but book records are materialized lazily. Indexers should start from block `41907604`, ingest `MarketCreated`, and enrich market, token, and book state through diamond read functions.
